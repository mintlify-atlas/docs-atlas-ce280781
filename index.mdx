Connect your own OpenAI, Gemini, or Claude API keys, install the GitHub App or configure a GitLab webhook, and every PR gets a thorough review automatically posted as a comment.

## How Lynx works

Lynx runs as a self-hosted service alongside your existing Git platform. When a developer opens a pull request, Lynx receives a webhook event, fetches the diff, and sends it to each of your configured LLM providers simultaneously. The resulting reviews are posted back as comments on the PR.

<Steps>
  <Step title="Deploy Lynx">
    Run Lynx with Docker Compose on your own server. Choose between a local development setup or a production deployment with HTTPS via Traefik.
  </Step>
  <Step title="Connect your Git platform">
    Install the Lynx GitHub App on your repositories, or configure a GitLab webhook to start receiving PR/MR events.
  </Step>
  <Step title="Add your LLM API keys">
    Log in to the Lynx dashboard and add API keys for OpenAI, Gemini, and/or Claude. Keys are encrypted at rest using Fernet encryption.
  </Step>
  <Step title="Open a pull request">
    Lynx automatically posts AI review comments on every new PR. You can also trigger a manual review by commenting `/review` on any PR.
  </Step>
</Steps>
